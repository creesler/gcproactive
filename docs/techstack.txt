Build a simple MVP web app for my automation agency.
Goal:
Create a client dashboard plus admin dashboard for a prepaid credit conversion system. This system converts third-party provider usage into our own internal credits. Clients will use their own n8n instance, and their n8n workflows will POST usage events to our Node.js API. Our server will compute billable credits using provider pricing plus our markup rules, deduct balance, and store logs.
Important:
This is only for the prepaid system and monthly service tracking. Do not build workflow editing, chatbot builders, advanced analytics, or provider-native dashboards.
Core product logic:
1.	Clients have a prepaid credit balance.
2.	Client n8n workflows send usage events to our Node server webhook after billable provider calls.
3.	Our Node server converts usage into raw cost.
4.	Our Node server applies markup rules.
5.	Our Node server converts billable amount into internal credits.
6.	Credits are deducted immediately.
7.	Every event is logged.
8.	Monthly hosting/retainer/support tracking is separate from prepaid credits.

Support these formulas:
1.	raw_cost based:

2.	unit based:
raw_cost = quantity * provider_unit_price
credits_used = ceil((raw_cost * markup_multiplier + fixed_surcharge) / credit_value)
3.	token based:
raw_cost = (input_tokens / 1000000 * input_price) + (output_tokens / 1000000 * output_price)
credits_used = ceil((raw_cost * markup_multiplier + fixed_surcharge) / credit_value)
